# 当前进度

## 已完成 ✅

### 核心功能 v1.0.0 (2026-05-18)

- [x] **独立 Playwright 库版本**：完全脱离 Claude Code Agent，零 Token 消耗
- [x] **UI 模式**：通过 Playwright CDP 连接 Chrome，浏览器自动化操作
- [x] **API 模式**：HTTP 直调启小铺 API，每单处理 <500ms
- [x] **混合模式**：浏览器提取订单 + API 操作订单（高效 + 可靠）
- [x] **GUI 控制台**：tkinter 桌面应用，启动/暂停/停止/实时日志
- [x] **留言解析器**：4 个白名单关键词，保守跳过策略

### 安全机制

- [x] **改价验证**：改价后强制 DOM 验证金额（容差 ±0.01）
- [x] **污染检测**：改价前读取服务端真实值，校验 productTotal 未被污染
- [x] **备用金规则**：含"备用金"的订单改价后不确认付款
- [x] **连续失败保护**：连续 3 单失败自动停止，写告警文件
- [x] **运行锁**：防止并发执行
- [x] **超时保护**：软超时 10 分钟 + 硬超时 11 分钟
- [x] **上限控制**：单次最多处理 15 单

### 运维设施

- [x] **日志系统**：带时间戳、跨 0 点自动切换文件
- [x] **日志归档**：超过 2 天的日志自动移入 YYYY-MM/ 目录
- [x] **每日汇总**：统计前一天处理量
- [x] **告警文件**：异常写入 `data/alerts.txt`，500KB 自动轮转
- [x] **异常截图**：改价失败/弹窗未关闭时自动截图
- [x] **控制文件**：`data/control.json` 在线切换模式
- [x] **常驻模式**：`--loop` 参数，每 5 分钟自动运行
- [x] **演习模式**：`--dry-run` 只读不写

### 文档

- [x] **SKILL.md**：技能说明
- [x] **rules/**：3 份规则文档
- [x] **docs/changelog.md**：更新日志
