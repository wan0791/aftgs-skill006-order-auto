# 改价计算详解

## 核心公式

```
差价 = 目标价格 - 当前实际支付金额
```

**改价窗口填的是差价，不是目标价格。**

## 示例

订单：青石斑鱼 ¥32.00 ×1，运费 ¥28.00
当前实际支付：¥60.00
买家留言："改价30"

```
目标价 = 30.00（从留言提取）
差价   = 30.00 - 60.00 = -30.00

在改价输入框中填入：-30
```

## 价格提取规则

### 有序匹配（优先级从高到低）

1. `改价[改] XXX 元` — 如 "改价30元"、"改价改30元"
2. `改价XXX` — 如 "改价30"
3. `备用金 XXX 元` — 如 "备用金50元"
4. `备用金XXX` — 如 "备用金50"
5. `XXX 元` — 兜底匹配

### 浮点数容差

`abs(当前价 - 目标价) <= 0.01` → 视为已匹配，跳过操作。

## 运费处理

- 当前版本不自动修改运费，维持原运费
- 如留言明确要求"免运费"，需手动处理
