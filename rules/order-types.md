# 订单类型详解

基于技能 004（微分销）规则，适配启小铺。

## 白名单关键词

```
改价 | 备用金 | 直接确认 | 确认付款
```

**其余内容一律视为噪音**：
- "叫车"、"叫跑腿"、"装车" — 物流相关，跳过
- "试菜金"、"已转财务" — 财务相关，跳过
- "随货发"、"删单"、"重下" — 操作相关，跳过
- "藕盒只发1件"、"肚条14" — 备注信息，跳过
- 所有电话号码、纯数字 — 非指令，跳过

## 类型判定流程

```
留言为空 / "留言" / 图标？
  ├─ 是 → 类型1：跳过
  └─ 否 → 包含"备用金"？
            ├─ 是 → 有金额？
            │         ├─ 是 → 金额=当前？ → 类型4：跳过
            │         │         └─ 金额≠当前？ → 类型3：改价 + 跳过确认
            │         └─ 否 → 类型0：跳过
            └─ 否 → 包含"改价"？
                      ├─ 是 → 有金额？
                      │         ├─ 是 → 金额=当前？ → 类型2a：跳过
                      │         │         └─ 金额≠当前？ → 类型2：改价 + 确认
                      │         └─ 否 → 类型0：跳过
                      └─ 否 → 包含"直接确认"/"确认付款"？
                                  ├─ 是 → 类型5：直接确认
                                  └─ 否 → 类型0：跳过
```

## 启小铺适配说明

- 界面、按钮 CSS 类名与微分销完全一致
- 待付款页 URL 同为 `/Order/lists/type/0/status/0/`
- 弹窗组件均为 `.jbox`
- 留言解析逻辑 100% 复用技能 004
